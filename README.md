# 🌸 Todo Petal – Task Management System

A modern desktop-based productivity application built using **Java Swing** and **MySQL** that helps users efficiently manage tasks, track goals, organize schedules, and improve daily productivity.

**Nafeesathul Misriya** & **Ishan Thomas Eapen**

## 📌 Project Overview

Todo Petal is an all-in-one productivity platform designed to simplify task management through a clean, intuitive, and user-friendly interface.

The application enables users to:

* Manage daily tasks efficiently
* Track personal and professional goals
* Monitor deadlines and schedules
* Organize important information securely
* Improve productivity through structured planning

Developed as part of the **B.Tech Computer Science and Engineering Curriculum**, the project demonstrates practical implementation of software engineering principles, GUI development, database management, and object-oriented programming concepts.

## 🚀 Installation & Setup

### Prerequisites

* Java JDK 17+
* MySQL Server
* MySQL Connector/J

### Clone Repository

```bash
git clone https://github.com/MISRIYA-7/TodoPetal-Task-Management-System.git
cd TodoPetal-Task-Management-System
```

### Create Database

```sql
CREATE DATABASE topetal;
USE topetal;
```

### Configure Database Connection

```java
String url = "jdbc:mysql://localhost:3307/topetal";
String username = "your_username";
String password = "your_password";
```

### Run Application

```bash
javac *.java
java Main
```
